# I321_distributed_systems_Alex_Alexandre
Systèmes distribués projet de groupe Alexandre Ramirez Alex Kamano

# Procédure d'installation complète

## Pré-requis
Avant de commencer, assurez-vous de disposer 
1. de Git installé sur votre machine.
2. d'un IDE (WebStorm ou VSCode) *conseillé*

## Procédure
Ouvrez un terminal ou depuis l'IDE choisi
exécutez la commande : 
<br>
```git clone https://github.com/Alexandre-Ramirez/I321_Alex_Alexandre_distributed_systems.git```
<br>
ou un clone directement via votre IDE 

### Installation des dépendances et vérification des versions

**Installation des dépendances**
Dans le dossier du projet :
<br>
```npm install```
<br>
Cette commande installe Express, Pug, Morgan, dotenv et l’ensemble des dépendances nécessaires.

**Vérification des versions**
<br>
commandes : ```npx -vnpm -vnode -v```
<br>
résultat : ```npx : 11.6.2 npm : 11.6.2 node : v22.18.0```
